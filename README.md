Return Function Example

A beginner-friendly Python project that demonstrates how to use the return statement inside functions.

📌 Description

This program:

Creates a function named add()
Takes two numbers as parameters
Returns the sum of the numbers
Stores the returned value in a variable
Prints the final result

🧠 Concepts Used

Functions
Function Parameters
return statement
User Input
Integer Conversion using int()
Variables
Addition Operator

💻 Code

def add(a, b):
    return (a + b)

a = int(input("enter your num:"))
b = int(input("enter your num:"))

result = add(a, b)

print(result)
